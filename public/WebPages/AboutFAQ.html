Project Overview

Atlas of Ancient Civilizations

This project began as a university team build and has been upgraded into a cleaner interactive history explorer that combines historical map overlays with short contextual narratives.

The goal is simple: make large-scale history easier to browse visually. Visitors can compare major ancient powers, inspect geographic boundaries, and read concise notes without leaving the map.

For a portfolio audience, the project now highlights practical full-stack work: Express routing, MySQL-backed content, authentication, local persistence for preferences, and data-driven rendering with Leaflet.

Frequently Asked Questions

What is this project about?

It is an interactive map experience focused on ancient civilizations, designed to make history feel visual, approachable, and easier to compare across regions.

What technologies power it?

The application uses Node.js, Express, MySQL, vanilla JavaScript, and Leaflet for interactive geographic overlays.

What was improved after the original class submission?

The codebase was cleaned up, repeated logic was refactored into reusable modules, UI copy was rewritten, and the app structure was made more maintainable for long-term showcase use.
